# 🌬️ Wind Portfolio Intelligence

Finance-grade analytics application for exploring and comparing wind energy project portfolios.  
The tool enables scenario analysis, KPI tracking, and risk assessment across projects, supporting better investment and planning decisions.

---

## Key Features
- Portfolio KPIs: Capacity, IRR, NPV  
- Scenario adjustments: CAPEX, yield, PPA  
- Risk analysis: DSCR, health scoring  
- Project-level drill-down (20-year view)  
- Side-by-side project comparison
